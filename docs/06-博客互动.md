# Plan 06：博客互动

> 对应计划：[`.cursor/plans/06-博客互动.md`](../.cursor/plans/06-博客互动.md)
>
> **交付日期**：2026-06-30  
> **架构形态**：模块化单体（`internal/blog/{repo,service}` + `internal/handler`）

## 交付摘要

迁移 Nest `comment` / `reply` / `like` / `collect` 至 Go 单体：C 端评论/回复/点赞/收藏与管理端评论审核接口可用；评论创建接入 Plan 04 敏感词过滤与命中记录；文章列表 `commentCount` 由评论表统计回填。

## 目录与模块

| 路径 | 职责 |
|------|------|
| `internal/blog/repo/comment_repo.go` 等 | comment/reply/like/collect Ent 访问 |
| `internal/blog/service/comment_service.go` 等 | 限流、敏感词、通知、幂等点赞/收藏切换 |
| `internal/blog/service/sensitive_content.go` | 敏感词过滤共用逻辑 |
| `internal/handler/comment_handler.go` 等 | HTTP 端点 |
| `internal/user/sensitive/service.go` | 命中审核同步 comment/reply/msgboard 状态 |
| `deploy/postman/interaction-smoke.json` | M3 互动域冒烟 |

## 配置与环境

无新增必填配置；评论/留言限流复用 Redis（`comment:rate:*`、`msgboard:rate:*`）。

## 接口一览

### `/api/v1/comment/*`

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/comment/create` | 创建评论（JWT） |
| DELETE | `/comment/delete?id=` | 删除评论（JWT） |
| GET | `/comment/findAll` | 文章评论列表（公开，含嵌套回复） |
| GET | `/comment/admin` | 管理端评论列表 |
| GET | `/comment/my-list` | 我的评论（JWT） |
| GET | `/comment/on-my-articles` | 作者收到的评论（JWT） |

### `/api/v1/reply/*`

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/reply/create` | 创建回复（JWT） |
| DELETE | `/reply/delete?id=` | 删除回复（JWT） |
| GET | `/reply/findAll` | 评论下回复（query `articleId` 实为 parentId，与 Nest 一致） |
| GET | `/reply/my-list` | 我的回复（JWT） |

### `/api/v1/like/*`

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/like` | 点赞/取消（body: `articleId`, `status`） |
| GET | `/like/check` | 是否已点赞（JWT） |
| GET | `/like/my-ids` | 已点赞文章 ID 列表（JWT） |

### `/api/v1/collect/*`

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/collect` | 收藏 toggle（JWT） |
| DELETE | `/collect/:id` | 按记录 id 取消（JWT） |
| GET | `/collect/list` | 收藏列表（JWT） |
| GET | `/collect/check` | 是否已收藏（JWT） |
| GET | `/collect/count` | 文章收藏数（公开） |

## 与 NestJS 差异

| 项 | 说明 |
|----|------|
| RPG 事件 | 无 `COMMENT_CREATED` / `LIKE_CREATED` 等（Plan 09） |
| avatarFrame | 评论/回复 userInfo 不含 RPG 头像框 |
| BanGuard | 未实现 RPG 封禁拦截 |
| 点赞幂等 | 与 Nest 一致：重复 POST 可插入多条；取消删首条 |
| 错误码 | HTTP 200 + `code` 字段，429 限流与 Nest 语义对齐 |

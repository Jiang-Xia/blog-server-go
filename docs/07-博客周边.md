# Plan 07：博客周边

> 对应计划：[`.cursor/plans/07-博客周边.md`](../.cursor/plans/07-博客周边.md)
>
> **交付日期**：2026-06-30  
> **架构形态**：模块化单体（`internal/blog/{repo,service}` + `internal/handler`）

## 交付摘要

迁移 Nest `msgboard` / `link` / `file` / `resources` 模块：留言板、友链、文件上传与资源库 CRUD 可用；邮件验证码沿用 Plan 02 `internal/user/email`；静态资源通过 Hertz Static 挂载 `/static/` → `storage.upload_path`。

## 目录与模块

| 路径 | 职责 |
|------|------|
| `internal/blog/repo/msgboard_repo.go` 等 | msgboard/link/file Ent 访问 |
| `internal/blog/service/msgboard_service.go` 等 | 留言限流、敏感词、友链 CRUD |
| `internal/blog/service/resources_service.go` | 上传、分片合并、代理接口 |
| `internal/handler/msgboard_handler.go` 等 | HTTP 端点 |
| `pkg/config/config.go` | `storage.upload_path` / `public_prefix` |
| `configs/monolith.yaml` | 存储路径默认值 |
| `deploy/postman/blog-peripheral-smoke.json` | M3 周边域冒烟 |

## 配置与环境

```yaml
storage:
  upload_path: "./public/uploads/"
  public_prefix: "/static/"
```

- 磁盘路径与 Nest `./public/uploads/` 对齐
- URL 映射：`/static/...`（Nest 为 `/static/` 前缀）
- 邮件：复用 `mail.*` SMTP；未配置时验证码接口返回错误（与 Plan 02 一致）

## 接口一览

### `/api/v1/msgboard/*`

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/msgboard` | 创建留言（公开，IP 日限 10 条） |
| GET | `/msgboard` | 留言列表 |
| POST | `/msgboard/delete` | 批量删除（JWT，body: id 数组） |

### `/api/v1/link/*`

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/link` | 申请友链 |
| GET | `/link` | 列表（`?client=` 仅已同意） |
| GET | `/link/:id` | 详情 |
| PATCH | `/link/:id` | 更新 |
| DELETE | `/link?id=` | 删除（JWT） |

### `/api/v1/file/*`

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/file/uploadBigFile` | 分片上传（JWT） |
| POST | `/file/uploadBigFile/merge` | 合并分片（JWT） |
| GET | `/file/uploadBigFile/checkFile` | 续传检查（JWT） |

### `/api/v1/resources/*`

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/resources/daily-img` | Bing 壁纸代理 |
| GET | `/resources/weather` | IP 卡片代理 |
| POST | `/resources/uploadFile` | 管理端上传（JWT） |
| POST | `/resources/upload-media` | 头像/封面/正文图（JWT） |
| POST | `/resources/upload-media/register-avatar` | 注册头像（公开） |
| GET | `/resources/files` | 文件分页列表 |
| GET | `/resources/register-avatars` | 注册可选头像 |
| GET | `/resources/file/:id` | 文件详情 |
| DELETE | `/resources/file?id=` | 删除（JWT） |
| POST | `/resources/folder` | 创建虚拟文件夹 |
| PATCH | `/resources/file` | 更新元数据 |

## 与 NestJS 差异

| 项 | 说明 |
|----|------|
| 百度统计 | `/resources/baidutongji` 未迁移（需 OAuth token 配置） |
| IP 归属 | 留言 IP 解析简化为 UA 启发式，未调用 vvhan geo API |
| 定时邮件 | 日报/友链健康/敏感词告警邮件 cron 未实现（Plan 08 scheduler） |
| resources 表 | Nest 实际使用 `x_file`，Go 与之对齐，无独立 resources 表 |

# Plan 02：认证与用户登录

> 对应计划：[`.cursor/plans/02-认证与用户登录.md`](../.cursor/plans/02-认证与用户登录.md)
>
> **交付日期**：2026-06-30  
> **架构形态**：模块化单体（`services/monolith/internal/user/`）

## 交付摘要

迁移 Nest 用户认证域至 Go 单体：账号/邮箱/微信登录、注册、刷新 token、GitHub OAuth、图形验证码、用户资料 CRUD、pub 统计 mock；响应体统一为 `{code:200, bizCode:200, message, data}`，与 blog-admin / Nest 对齐。

## 目录与模块

| 路径 | 职责 |
|------|------|
| `services/monolith/internal/user/auth/` | JWT、登录注册、OAuth、微信、密码校验 |
| `services/monolith/internal/user/captcha/` | 图形验证码 Redis 存储与频控 |
| `services/monolith/internal/user/email/` | 邮箱验证码 SMTP |
| `services/monolith/internal/user/profile/` | 用户资料 CRUD、列表 |
| `services/monolith/internal/user/repo/` | Ent 用户/角色/权限查询 |
| `services/monolith/internal/user/port.go` | `UserService` 接口（Plan 05 依赖） |
| `services/monolith/internal/handler/user_handler.go` | `/api/v1/user/*` 全量路由 |
| `services/monolith/internal/middleware/jwt_auth.go` | JWT 鉴权 |
| `services/monolith/internal/middleware/permission.go` | RBAC 权限（读 Redis/DB） |
| `services/monolith/internal/pub/` | `GET /pub/stats` mock |
| `pkg/crypto/` | bcrypt/PBKDF2/RSA |
| `deploy/postman/auth-smoke.json` | 契约冒烟 |

## 配置与环境

`configs/monolith.yaml` 新增/修正：

| 段 | 说明 |
|----|------|
| `app.blog_home` | OAuth 302 目标（如 `http://localhost:5050`） |
| `jwt.legacy_ttl` | 8h（兼容 token） |
| `jwt.access_ttl` | 30m |
| `jwt.refresh_ttl` | 168h（7d） |
| `crypto.rsa_private_key` | 空则用 Nest 开发默认密钥 |
| `oauth.github_*` | GitHub OAuth |
| `mail.*` | SMTP（未配置时 sendCode 返回 503） |
| `wechat.*` | 小程序登录 |

`services/monolith/cmd/main.go` 含 `//go:debug rsa1024min=0`（Nest 512 位 RSA 兼容）。

## 本地启动与验收

```powershell
cd d:\study\myGithub\blog-server-go
go run ./services/monolith/cmd/main.go

# 健康检查
curl.exe -s http://localhost:5000/api/v1/health

# 验证码
curl.exe -s http://localhost:5000/api/v1/user/authCode

# 需登录接口：默认超级管理员 18888888888 / super（与 Nest 一致）
go run scripts/dev_login.go --token-only

# Postman/newman
newman run deploy/postman/auth-smoke.json --env-var baseUrl=http://localhost:5000

# 单元测试
go test ./pkg/crypto/... -v
```

### 本地测试账号

| 字段 | 值 |

# Plan 05：文章内容

> 对应计划：[`.cursor/plans/05-文章内容.md`](../.cursor/plans/05-文章内容.md)
>
> **交付日期**：2026-06-30  
> **架构形态**：模块化单体（`internal/blog/{repo,domain,service}` + `internal/handler`）

## 交付摘要

迁移 Nest `article` / `category` / `tag` 模块至 Go 单体：C 端文章列表/详情/归档/统计与 admin 内容管理接口可用。作者信息通过 `user.UserService` 端口批量组装，**禁止 article repo 跨表 JOIN user/dept**（dept 名称单独查 `UserRepo.FindDeptByID`）。

## 目录与模块

| 路径 | 职责 |
|------|------|
| `services/monolith/internal/blog/domain/` | 列表/详情 DTO |
| `services/monolith/internal/blog/repo/` | article/category/tag Ent 访问 + `article_tags_tag` 维护 |
| `services/monolith/internal/blog/service/` | 用例逻辑、goldmark 补渲染 |
| `services/monolith/internal/blog/util/` | 随机分类/标签色 |
| `services/monolith/internal/handler/article_handler.go` 等 | HTTP 端点 |
| `services/monolith/internal/user/service.go` | `UserService` 端口 wire 注入 |
| `deploy/postman/article-smoke.json` | M3 文章域冒烟 |

## 配置与环境

无新增配置项；复用 `configs/monolith.yaml` 中 MySQL/Redis/JWT。

## 接口一览

### `/api/v1/article/*`（Nest ArticleController）

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/article/list` | 分页列表（body：`client`/`category`/`tags`/关键词） |
| GET | `/article/info` | 详情 + prev/next |
| POST | `/article/create` | 创建（JWT） |
| POST | `/article/edit` | 编辑（JWT） |
| DELETE | `/article/delete` | 物理删除（JWT） |
| POST | `/article/views` | 阅读量 +1 |
| POST | `/article/likes` | 点赞计数（兼容 Nest，主路径 Plan 06 `/like`） |
| PATCH | `/article/disabled` | 软删/恢复 |
| PATCH | `/article/topping` | 置顶 |
| GET | `/article/my-list` | 我的文章（JWT） |
| GET | `/article/archives` | 归档 |
| GET | `/article/related` | 相关推荐 |
| GET | `/article/author-stats` | 作者统计（JWT） |
| GET | `/article/statistics` | 数据大屏统计 |

### `/api/v1/category/*`、`/api/v1/tag/*`

与 Nest CategoryController / TagController 路径一致（CRUD + `GET /tag/:id/article`）。

## 本地启动与验收

```bash
# 启动（仓库根目录）
set CONFIG_PATH=configs/monolith.yaml
go run ./services/monolith/cmd/main.go

# 健康检查
curl -sf http://localhost:5000/health

# C 端文章列表
curl -sf -X POST http://localhost:5000/api/v1/article/list \
  -H "Content-Type: application/json" \
  -d "{\"page\":1,\"pageSize\":10,\"client\":true}"

# M3 文章域冒烟
newman run deploy/postman/article-smoke.json --env-var baseUrl=http://localhost:5000
```

## 与 NestJS 差异

| 项 | 说明 |
|----|------|
| 分类结构 | Nest 为**扁平列表**（非树形）；Plan 文档「树形 CRUD」未实现 |
| 评论数 | 列表 `commentCount` 暂为 0，Plan 06 接 CommentService |
| 统计趋势 | `statistics.trends` / `totalComments` 简化，Plan 06/08 补全 |
| RAG/事件 | 无 `BlogEventPublisher`（Plan 08） |
| avatarFrame | 详情不含 RPG 头像框（Plan 09） |
| Markdown | 主路径仍用客户端传入 `contentHtml`；空时 goldmark 服务端补渲染 |
| 作者过滤 | C 端用 `UserRepo.ListActiveUserIDs` 二次过滤，非 SQL JOIN |

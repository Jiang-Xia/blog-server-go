## 关键实现要点

### 跨模块调用（不跨表 JOIN）

```go
// internal/blog/service/article.go
func (s *ArticleService) Detail(ctx context.Context, id uint64) (*ArticleDetailDTO, error) {
    article, err := s.repo.GetByID(ctx, id)
    author, err := s.userSvc.GetUser(ctx, article.AuthorID)
    return &ArticleDetailDTO{Article: article, Author: author}, nil
}
```

### Ent repo 模式

- `internal/blog/repo/` — 仅访问 blog 域表
- `internal/blog/domain/` — 领域模型
- `internal/blog/service/` — 用例逻辑
- `internal/blog/handler/` — Hertz HTTP handler

### Markdown 渲染

- goldmark 渲染，对照 NestJS marked 安全配置防 XSS

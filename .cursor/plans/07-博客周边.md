## API 路径（保持兼容）

| 路径 | 说明 |
|------|------|
| `/api/v1/msgboard/*` | 留言板 |
| `/api/v1/link/*` | 友链 |
| `/api/v1/file/*` | 文件上传 |

## 关键实现要点

### 文件上传

- Hertz multipart，存储路径与 NestJS 一致
- 静态资源访问路径保持 `/uploads/*` 或现有 CDN 规则

### 邮件

- gomail 发送（注册验证、通知邮件等）
- SMTP 配置与 NestJS 对齐

### 敏感词

- msgboard 创建时调用 Plan 04 敏感词过滤

## 风险与注意点

| 风险 | 对策 |
|------|------|
